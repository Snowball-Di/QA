# BIT_QA_System

## 后端

后端语言为Python，要运行服务器，请执行：

```shell
cd QA/backend/
python3 server.py
```

也可在本地使用命令行直接启动问答：

```shell
cd QA/backend/
python3 interactive_qa.py
```

所需运行环境：

python>=3.8

pytorch==1.9.1

transformers==4.12.5

scipy==1.6.2

flask==1.1.2

flask_cors==3.0.10

OpenCC

numpy

scikit-learn

colorama
